## 👋 About Me

- 요구사항을 기반으로 **사용자 경험을 고려한 안정적인 서비스 개발**을 지향합니다.
- 문제의 원인을 분석하고 **구조적으로 해결하는 과정**을 중요하게 생각합니다.
- 새로운 기술을 즐기며, 이를 **기록으로 남기고 협업을 통해 함께 성장하는 개발자**입니다.
